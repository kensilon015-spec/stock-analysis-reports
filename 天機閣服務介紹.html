個股深度研究報告

Service 01
給一個股票代號（台股或美股），產出一份從資料蒐集到投資結論的完整分析報告——比法人報告更全面，因為同時檢驗多空雙方的證據。
六大管道資料蒐集
管道蒐集內容
財務數據近 5 年營收、毛利率、營業利益率、EPS、自由現金流、ROE / ROA、負債比
法人籌碼三大法人買賣超、外資持股比例變化、融資融券餘額、大股東申報轉讓
新聞輿情近期重大新聞、媒體報導方向、PTT / 論壇討論熱度與情緒
法說會最近一次法說會重點摘要、管理層展望與語氣變化
年報分析營收結構拆解、主要客戶與供應商集中度、研發投入佔比、關係人交易
產業報告所屬產業趨勢、市場規模成長率、技術替代風險、國際同業動態
分析流程亮點
九步遞進分析
從原始資料彙整 → 財務情報分析 → 產業競爭力 → 多空假說驗證 → 壓力測試 → 品質評審 → 完整報告 → 情境推演 → 投資結論，每一步都建立在前一步的基礎上。
四次信心校準
ACH 多空對決、壓力測試最壞情境、23 項逐項評審、推演路徑模擬——四道關卡層層篩選，防止自己騙自己。
貓膩探索
專門用反向思維檢視「市場共識可能錯在哪裡」、「管理層可能隱瞞什麼」、「財報數字背後有沒有不尋常的地方」。
具體結論
不給模糊的「可以關注」，而是明確給出：買入 / 觀望 / 迴避，附建議價位區間與翻轉條件。

產業鏈透視

Service 02
從一家公司出發，展開整條供應鏈地圖——找出誰是關鍵節點、誰有替代風險、一個環節出問題會牽動哪些公司。
五大功能
供應鏈上下游展開
自動展開 3 層：上游供應商 → 本公司 → 下游客戶。標示供貨佔比、替代難度、合作年限，找出「單一供應商依賴」的致命節點。
同業競爭矩陣
橫向比較同產業 5-10 家公司的營收規模、毛利率、ROE、市佔率、本益比，一張表看出龍頭、高估股、低估股。
概念股篩選
輸入題材（如「AI 液冷散熱」），列出所有相關公司並依受惠程度分級——核心受惠、間接受惠、邊緣沾光，避免買到假概念股。
骨牌效應模擬
模擬「A 公司出問題，誰會受傷」的連鎖反應。也能反向追蹤：某公司股價異常，追溯可能的上游事件源頭。
三方驗證法（確保關係真實）
每一條供應鏈關係都必須通過至少 2/3 來源驗證，避免建立「聽說」等級的虛假關係：
來源內容
來源 A年報揭露——公司自行申報的客戶 / 供應商名單
來源 B公開新聞 / 法說會——媒體報導或管理層提及的合作關係
來源 C產業報告 / 第三方資料庫——獨立研究機構的交叉確認

弱訊號偵測（核心差異化能力）
運作原理：單一事件可能不起眼，但多個微小異常同時指向同一方向時，就是趨勢即將形成的訊號。

範例：

A
供應商法說會提到「交期延長兩週」
單獨看 → 可能只是個案，記錄追蹤
B
同業悄悄調高報價 3-5%
單獨看 → 可能只是成本轉嫁，記錄追蹤
C
下游客戶開始建立安全庫存
單獨看 → 可能只是季節性備貨，記錄追蹤
↓ 7 天內 3 個弱訊號收斂 ↓
!

觸發深度分析 → 找出受惠股（有備貨的）與受害股（依賴單一來源的）

三大服務如何串聯

每日情報掃描 → 偵測到個股異常 → 個股深度研究
每日情報掃描 → 偵測到產業趨勢 → 產業鏈透視 → 鎖定標的 → 個股研究
個股深度研究 → 需了解上下游 → 產業鏈透視
弱訊號收斂 → 確認影響範圍 → 產業鏈 + 個股研究
不是三個獨立工具，而是一個持續運轉的情報循環。
每日情報偵測持續餵養資料，異常觸發深度研究，研究過程需要產業鏈支撐，產業鏈又反過來發現更多值得追蹤的標的。
