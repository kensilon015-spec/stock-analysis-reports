關鍵節點：泰國新廠是否能在 Q2 如期投產並在 Q3 達到 70%+ 產能利用率。如果延遲 1-2 季，營收跳增的時間點會推遲到 2026H2 才顯現。

脆弱度評估：中等。臺達電的追單是真實的，但「2.5 倍成長」中有多少比例落到經寶，公司刻意不透露具體數字。如果落到經寶的只是 30-40%，而非市場期待的 50%+，增幅會令人失望。

推演鏈 2：CB 稀釋衝擊鏈（空方路徑）

股價 > 轉換價 → CB 持有人轉換 → 股本膨脹 → EPS 被動壓縮 → PE 升高 → 估值壓力
CB3（3.015 億，3 年期）2026/03 增資完成 → 歷史三次 CB 全部 100% 轉換（硬事實） → 股本從 50.9M → 57.3M 股（+12.6%，+6.4M 股） → 即使獲利不變，EPS 從 ~12 元壓縮到 10.55 元 → PE 從 ~22x 被動升到 25.9x → 估值吸引力大幅下降

關鍵節點：轉換時間點。如果 CB 持有人在股價高檔（250+）時集中轉換，市場賣壓會在 2-3 個月內集中釋放。歷史上 jpp 二的轉換集中在到期前 6 個月。

白話說：公司用你的錢去擴張，成長的果實被稀釋掉了。你看到營收 +50%，但 EPS 只成長 20% 出頭，少掉的那 30% 被新股東拿走了。這是三次 CB、三次增資的代價。

推演鏈 3：SEGNERE 整合鏈（雙向路徑）

正向：司法重整收購 → 客戶回穩 → 國防切入 → 歐洲營收翻倍
160 萬歐元收購 SEGNERE（司法重整公司） → 取得歐洲連接器客戶名單 + 國防認證基礎 → 客戶回穩、開始貢獻營收 → 切入歐洲國防供應鏈（長約、高毛利） → 歐洲營收從近乎零 → 年貢獻 2-3 億台幣
反向：整合失敗 → 客戶流失 → 虧損拖累 → 管理層注意力分散
司法重整公司本身有系統性問題（才會破產） → 客戶對新東家信心不足、轉單 → 整合投入的人力/資金超出預期 → 管理層注意力從泰國主戰場分散 → 年虧損 0.3-0.5 億台幣 + 泰國本業爬坡延遲

關鍵節點：2026/09（整合滿一年）是判斷正向或反向的分水嶺。如果屆時 SEGNERE 仍無具體營收貢獻或國防訂單，反向路徑的機率將大幅上升。

8c 整合結論

三條鏈整合起來看：多方有一條主力鏈（AI 液冷），一條潛力鏈（SEGNERE），但空方的 CB 稀釋鏈幾乎是確定的（歷史 100% 轉換）。

白話說：經寶的故事是「成長是真的，但成長的果實不完全屬於你」。AI 液冷帶來的營收增長，有一部分被 CB 稀釋吃掉了。而 SEGNERE 目前是薛丁格的貓——可能是驚喜也可能是地雷，11 個月後才能打開箱子看。

所以呢：推演鏈中最脆弱的節點是「泰國新廠投產時間」和「SEGNERE 整合進度」。兩者都缺乏公開資訊可驗證，這本身就是風險。而 CB 稀釋是唯一確定會發生的事件——投資人必須把它當作「已發生」來計算估值。

推論 → 結論：期望報酬為負 18%，這不是一個適合現價進場的標的。除非你有非常強的理由相信路徑 A 的機率 > 50%（例如你有管道確認臺達電給經寶的訂單佔比），否則不應該在 270 以上買入。

基本面訊號
1. 泰國歷史 CAGR 僅 9% → 2025 +57% 是異常爆發，不是常態
目前狀態：泰國子公司過去 5 年營收 CAGR 約 9%，2025 年突然跳到 +57%。市場將此解讀為「AI 液冷帶動的結構性轉變」。
弱訊號：但歷史告訴我們，經寶的泰國營收從未維持過 >20% 的成長超過 2 年。2018-2019 的通訊基站景氣也曾帶來一波 +30%，隨後回落到個位數。
若放大可能產生的影響：如果 2026 年增速回落到 15-20%（回歸均值），市場會從「AI 成長股」重新定價為「週期性製造業」，PE 將從 25x 壓縮到 15-18x。以 EPS 10.55 計算，股價可能回到 158-190 元。
白話說：+57% 是特例還是新常態？這個問題的答案價值 100 元的股價差。
基本面訊號
2. 法說會刻意不透露臺達電毛利率 → 可能低於市場預期
目前狀態：公司在法說會被問到臺達電訂單的毛利率時，以「商業機密」為由拒絕回答。但其他客戶的毛利率範圍都有概略說明。
弱訊號：選擇性保密通常暗示數字不好看。如果臺達電是大客戶但毛利率低於公司平均（36%），那「營收成長但毛利不升」的現象就有解釋了。
若放大可能產生的影響：AI 液冷帶來的是「量增但利薄」的成長，長期看公司的獲利品質下降。毛利率可能被鎖定在 34-36% 而非市場期待的 37-39%。
白話說：不說的通常是壞消息。臺達電的議價能力遠超經寶，不排除經寶是「以價換量」。
產業訊號
3. SEGNERE 司法重整 + 160 萬歐元 → 便宜貨或地雷？
目前狀態：160 萬歐元（約台幣 5,500 萬）收購一家歐洲連接器公司。價格低到不尋常——這代表公司幾乎沒有資產價值，只剩客戶名單和認證。
弱訊號：司法重整公司通常有未清償的債務、法律糾紛、員工問題。160 萬歐元是「買門票」，真正的成本在後面的整合投入。如果整合需要追加 500-1000 萬歐元，投資回收期會大幅延長。
若放大可能產生的影響：最壞情境：SEGNERE 持續虧損 + 追加投資 + 管理層精力分散 → 對合併報表 EPS 的拖累可能達 0.3-0.5 元/年。最好情境：取得歐洲國防認證 → 開啟全新高毛利業務線。
白話說：便宜有便宜的道理。160 萬歐元像是買一張樂透——中了大獎（國防訂單），沒中就是丟水裡。
地緣/政策訊號
4. 川普若對泰國加關稅 → 經寶製造基地優勢瓦解
目前狀態：經寶 90%+ 產能在泰國，這原本是優勢（避開中美貿易戰）。但川普 2026 對等關稅政策範圍持續擴大，東南亞不再是安全港。
弱訊號：泰國對美國的貿易順差持續擴大（2025 約 380 億美元），在川普的邏輯中遲早會被「處理」。EEC 特區的出口免稅優惠也可能被美方視為「不公平補貼」。
若放大可能產生的影響：若對泰國加徵 10-25% 關稅，經寶的通訊基站出口（佔營收 56%）成本直接上升。客戶可能轉向墨西哥或越南供應商。這是路徑 D 的觸發器。
白話說：經寶把雞蛋全放在泰國籃子裡。現在籃子是安全的，但如果川普伸手過來，籃子就不安全了。
基本面訊號
5. 2024 呆帳客戶身份不明 → 如果是航太維修方，復甦可能反帶新風險
目前狀態：2024 年有一筆呆帳損失，但公司未揭露客戶身份。根據產品線推測可能是航太維修（MRO）領域。
弱訊號：如果呆帳客戶是航太 MRO 方，而 MRO 市場正在復甦（疫後航班恢復），經寶可能會重新接觸類似客戶。但同類客戶的信用風險可能相似。
若放大可能產生的影響：若再次發生呆帳（金額可能更大），會直接衝擊 EPS 0.5-1.0 元。而且呆帳損失在季報公布前不易偵測，屬於「突發性風險」。
白話說：不知道的客戶是最危險的客戶。如果公司不願意說，投資人就沒辦法評估。
8g 整合結論

五個弱訊號整合：最值得警惕的是 訊號 1（泰國 CAGR 歷史 9%）和 訊號 2（臺達電毛利率黑箱），因為它們直接挑戰了「AI 帶動結構性成長」的核心敘事。

弱訊號收斂方向：五個訊號中有 4 個偏空、1 個雙向（SEGNERE），而且它們指向同一個結論——「市場對經寶的成長預期可能過高」。這不是單一風險，是多個微小異常同時指向同方向的收斂。

天機閣洞察：「泰國 CAGR 9%」是最被忽略的數據。市場只看到 +57% 的爆發，沒有人回頭看這個成長率在歷史上的持續性。如果均值回歸發生在 2026H2，現在的 PE 25.9x 就是定時炸彈。

所以呢：這些弱訊號不需要全部成真——只要其中 2 個同時發生（例如「成長回歸 + 毛利不如預期」），就足以觸發路徑 C。將蝴蝶效應監控列為高優先。
