8d. 風險級聯分析

級聯效應路徑圖

級聯路徑 1：漲價停滯 → 毛利率壓縮 → PE崩塌

電子布漲價停滯 → 營收增速歸零 → 毛利率從27%回落至18% → EPS從1.64降至0.8 → PE從62x需壓縮至30x → 目標價24元（-76%）

級聯路徑 2：質押爆倉 → 強制賣出 → 融資追繳

股價跌破質押維持率 → 董事長被迫賣股 → 市場恐慌 → 融資追繳 → 斷頭賣壓 → 股價加速下跌（融資78,000張 × 追繳 = 巨大賣壓）

級聯路徑 3：供給海嘯 → 價格戰 → 虧損

2027H2日東紡3倍+巨石10萬噸投產 → 電子布價格大跌30-40% → 富喬毛利率回到0-5% → 再度虧損 → 股價回到2023年水準（15-25元）

三條級聯路徑互相強化 漲價停滯導致股價下跌 → 觸發質押爆倉 → 融資追繳加速下跌 → 恰好碰上供給海嘯 → 形成負面循環。這就是為什麼高質押+高融資+週期反轉的組合如此危險。

8h. 貓膩推衍
基於第七步D的貓膩判定（嚴重性78/100），推衍貓膩可能的演化路徑

貓膩推衍：三種演化路徑

路徑 M1：12月獲利暴跌為一次性費用（機率40%）

推衍：公司在年報中說明12月獲利暴跌為泰國廠前期費用/設備折舊/年度獎金一次性認列。

影響：緩解部分擔憂，但不改變估值過高的事實。股價可能反彈至110元後再度下跌。

後果：管理層誠信度從25/100微升至35/100，但仍低於50閾值。看空信心從95%降至88%。

路徑 M2：12月獲利暴跌反映結構性問題（機率35%）

推衍：年報揭露12月獲利暴跌源於原物料成本飆升/客戶要求降價/產品組合惡化，且此趨勢在2026Q1持續。

影響：「超級週期」敘事瓦解。市場重新定價，PE從62x壓縮至25-30x。

後果：股價快速修正至40-50元。管理層誠信度維持25/100。看空信心升至97%。

路徑 M3：管理層新醜聞/質押危機爆發（機率25%）

推衍：董事長內線交易案有新進展（如判刑/新案件）、質押維持率跌破、或現增特定人名單被揭露與董事長有關聯。

影響：公司治理危機，市場信心崩潰。

後果：股價可能跌至20-30元。此路徑一旦發生將不可逆。

貓膩推衍加權結果 三條路徑的加權目標價：110×0.40 + 45×0.35 + 25×0.25 = 66.0元（vs現價102元 = -35%）。即使最樂觀的貓膩路徑（M1一次性費用），股價上行空間也僅有限（+8%至110元），而下行風險巨大（M2-55%、M3-76%）。
