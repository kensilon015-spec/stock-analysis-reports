8c. 影響鏈分析（蝴蝶效應推衍）

降息蝴蝶效應

起點：央行降息 1 碼（2.0% → 1.75%）
→ 儲值金信託利率下降 → 利息收入減少 ~3,000 萬 → EPS 減少 ~0.4 元
→ 市場重新定價（PE 不變但 E 下降）→ 合理價從 43.6 降至 ~38
→ 配息跟著降（3.05 → ~2.7）→ 殖利率吸引力下降
→ 若同時北捷替代率上升 → 雙重打擊 → 市場恐慌 → 興櫃流動性枯竭 → 想賣賣不掉
終點：股價可能從 50 跌到 30-35 元（-30% 至 -40%），且過程中幾乎無法止損

A 反向蝴蝶（正面路徑）
起點：Apple Wallet 宣布整合悠遊卡
→ iPhone 用戶回流（+30% 日交易筆數）→ 媒體大幅報導品牌回春
→ 悠遊付因流量效應新增 100 萬用戶 → 跨境支付也因手機整合加速
→ 市場重估從「配息股」升級為「轉型成功」→ PE 從 10x 跳到 15x
終點：股價可能從 50 漲到 65-70 元（+30% 至 +40%）
但此路徑機率極低（< 10%），因為 Apple Wallet 整合無任何跡象。

路徑 D：黑天鵝（10%）

觸發條件：CBDC 加速上路（2027 正式試行）OR 重大資安事件（儲值金信託出問題）OR 政策面強制合併（與一卡通合併）

CBDC 情境：若央行 2027 年正式推行零售型 CBDC，嵌入手機感應支付，儲值卡的「預存→利息」模式將根本性被取代。利息收入可能在 2-3 年內歸零。EPS 從 4.2 跌到 < 1.0。

合併情境：參考新加坡 SimplyGo 案例，政策面可能推動悠遊卡與一卡通合併。對現有股東可能是正面（合併溢價 30-50%）也可能是負面（被稀釋或低價收購）。

此路徑為尾部風險，2028 前發生機率極低。但需列入長期觀察清單。

推演版最終結論

加權期望路徑：50%×A + 30%×B + 10%×C + 10%×D → 2028 年 EPS 中位數 3.0-3.5 元，合理價 ~35 元。

最可能劇本：EPS 從 4.2 元緩降至 3.0-3.5（2-3 年），穩定成為準公營配息股。像一間政府開的小吃店——不會倒、不會擴張、生意慢慢被搶走一些，但老客戶還會來。你買這家店的股份，就是為了每年領分紅，不是為了等它變連鎖店。

行動建議：
1. 現在：不買。50 元溢價 37%，風險回報不對稱
2. 2026/Q4 後：若路徑 A 確認 → 43 元以下小量（≤ 3%）
3. 若路徑 B 確認：等 30 元以下才考慮
4. 觀察清單：央行利率 / 北捷替代率 / 悠遊付用戶 / Apple Wallet / FY2025 利息佔比
