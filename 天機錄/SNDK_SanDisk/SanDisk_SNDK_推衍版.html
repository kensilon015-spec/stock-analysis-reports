天機閣推衍方法一 — 5條關鍵事件影響鏈

IC-1：Micron消費端下修 → NAND缺口收窄 → 毛利率見頂

事件：Micron FQ3下修PC/手機NAND需求10-15% F
→ 一階：消費級SSD需求增速從+12%降至+5% 80%
→ 二階：供需缺口收窄5pp→2-3pp，Q2合約價漲幅收窄至15-25% 60%

IC-2：Samsung歷史性策略反轉 → NAND價格崩跌

觸發：Samsung歷史上3次在高價期後反轉（2008/2012/2018） I
→ 一階：Samsung增加NAND投片量10-20%或停止提價 25%（6M內）
→ 二階：合約價3-6月內跌20-35%，毛利率回落至35-45% 20%
→ 三階：EPS從$87→$30-45，Forward PE重估至15-20x，股價-35-50% 15%
核心風險 Samsung的NAND策略是所有路徑中最具決定性的單一變數。目前Samsung因HBM利潤更高而不攻NAND，但若HBM飽和，邏輯將反轉。Samsung Q1法說會（4月下旬）是最關鍵的觀察窗口。

IC-3：Stargate企業SSD認證成功 → 估值重估

事件：128TB QLC PCIe Gen5 SSD通過認證並出貨 I

→ 三階：估值從「大宗商品股」轉為「AI儲存成長股」，PE 12-15x 30%
交互作用 與IC-1互補：企業SSD成長可對沖消費端萎縮，降低營收組合風險。這是區分路徑1（超級週期）和路徑2（中等成長）的關鍵變數。

IC-4：AI CapEx持續超預期 → NAND估值框架轉變

持續中：超大規模客戶AI CapEx合計超$2,500億/年 I

→ 三階：NAND從週期大宗商品重新定位為AI基礎設施必需品 25%
注意 AI CapEx越強，Samsung越不可能攻NAND（HBM利潤更高）。但DeepSeek等開源模型可能降低AI推論硬體需求，長期影響待觀察。歷史先例：2000年.com泡沫電信CapEx持續上調直到突然停止。

七、個人化推衍
天機閣推衍方法三 — 依閣主情境的具體行動建議

情境A：已持有（成本$400-700）· 穩健型

短期（0-3個月）

  • 不加碼：$710接近分析師中位目標價$710，高於預期值$664
  • 設停損：$550-600（較現價-15%至-22%）
  • 等FQ3：最關鍵驗證事件，資訊不完整時不做重大決定

中期（3-6個月）— FQ3後決策

  • 超預期+企業SSD出貨 → 持有上看$800-900，可加碼至組合5%
  • 符合預期 → 持有，設獲利了結點$780-800
  • 低於預期 或 毛利率<60% → 減持30-50%
  • Samsung任何增產信號 → 立即減持50%

情境B：觀望中 · 穩健型

核心建議：不追高，等FQ3

  • $710是分析師中位目標價，追買的風險報酬比不佳
  • 可考慮小量試單：組合1-2%探路

FQ3後決策樹

FQ3結果行動
超預期 + 企業SSD出貨追買至組合3-5%
符合預期等回檔$600-650再建倉
低於預期放棄此波，等路徑3底部$420-500
不建議的行為 (1) $710+重倉追買 (2) 做空SNDK（短期催化劑偏多） (3) 因「低Forward PE」認為便宜——週期股PE是陷阱指標

情境C：積極型 · 高風險承受

策略：FQ3財報事件交易

  • 財報前2-3週小量建倉（組合2-3%）
  • 超預期 → 加碼至組合5-7%，目標$900-1,000
  • 符合預期 → 持有觀察，不加碼
  • 低於預期 → 停損出場，虧損控制在2%組合以內
  • 進階：選擇權價差策略（Bull Call Spread $700-$900）降低風險

風險控制：任何單一部位不超過組合10%，SNDK為高波動性個股
