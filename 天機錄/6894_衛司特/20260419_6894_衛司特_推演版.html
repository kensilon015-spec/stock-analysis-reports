8c

影響鏈分析

天機閣推衍方法一 — 事件 → 一階 → 二階 → 三階

鏈 1 · Q1 季報毛利率路徑分岔

Q1 2026 毛利率（5/15 公告） → 若 ≥ 36%（H1 路徑） → 市場認證「2024Q4 驟降僅為單季雜訊」 → PE 25× 維持／甚至擴張至 28× → 股價測試 420~450

→ 若 32~36%（H2 路徑 · 最可能） → 市場無新資訊、維持觀望 → PE 維持 25×、等待 Q2 再驗證 → 區間整理 360~400

→ 若 < 32%（H3 路徑） → 2024Q4 驟降被確認為結構性（銥金屬成本擠壓／新廠折舊攤提） → PE 被砍至 18~20×、EPS 下修至 12 → 急殺至 220~280（-30%~-45%）

鏈 2 · 台積電 2nm 擴產節奏

台積電 2025/Q4 法說會承諾 2nm 量產節奏 → 若 Arizona P1 2027 如期量產 → 衛司特 BOO 新約 2026H2 簽訂（5~8 億／年） → 2027 營收 +30% 確定 → 估值鎖定長線上漲動能

→ 若延後至 2028（情境 B） → 衛司特新約推遲、2027 營收成長僅 +10% → 市場失望、PE 修正 → 2028 成長斷層風險

鏈 3 · 三重頂部 + 稀釋預期 + 三變數重疊（看空放大鏈）

2028~2029 三變數同時成熟：① CB 轉換（歷史稀釋偏好 + 3 億元 CB）② 韓國 BOO 45% 股權邊界（無法完全控制）③ De Nora 續約（上游議價） → 任一惡化即觸發 → 三重頂部破下（380 元支撐失守） → PE 降至 18× + EPS 下修 → 極限下行 240~280（-30%~-38%）
跨鏈交互作用

鏈 1（Q1 季報）是短期催化劑、鏈 2（台積電）是中期錨定、鏈 3（三變數）是長期尾部風險。
三鏈同時看空的機率僅 ~8%（相互獨立假設下 0.21 × 0.40 × 0.10），但任一觸發都會使其他兩鏈的空頭機率提升 5~10pp（情緒連動）。

個人化推衍 · 分情境行動建議

依持股狀態 / 風險偏好 / 時間軸適配
情境 A · 空手者 · 觀望中

建議：等 Q1 季報後再進，避免在三重頂部接刀。

進場條件：① 現價回到 340 以下（對應 H2 中性下緣）且量能放大 → 首批 1/3 倉位；② Q1 毛利率 ≥ 36% 且股價已站上 400 → 追第二批 1/3；③ 站穩 420 後第三批 1/3。

停損：一旦跌破 280（加權期望 -26%）全出。
目標價位：420（H1 情境）／450（若 Q2 再驗證）。

情境 B · 現有持倉 · 成本偏低（< 300）

建議：分批減碼、鎖利 2/3、保留 1/3 長線部位。

減碼節奏：① 現價 390 減 1/3；② 若反彈至 400~410 減第 2 批；③ 若 Q1 後突破 420 暫停；④ 若跌破 380 啟動全面減倉。

核心邏輯：6 週漲 52.7%、加權期望 -10.8%，短線風險報酬比不佳；長線 BOO+接班故事仍成立，保留底倉等低點再加。

情境 C · 現有持倉 · 成本偏高（> 380）

建議：減碼到「心理舒適」水位後長抱，等 Q2 再評估。

行動：① 若能接受 -10~-15% 虧損上限 → 跌破 350 全出；② 若看好長線 → 現價減 1/2、剩下續抱但設 300 為硬停損。

不建議：向下攤平（在 H3 機率 21% 的情況下，攤平可能放大損失）。

情境 D · 避險需求（對沖）

建議：若有券商期貨／選擇權管道，可買 Put 或期貨空單對沖 1/3~1/2 倉位。

成本：3 個月價外 Put 權利金約 3~5% 倉位價值，相對於 H3 路徑 -30% 的潛在損失，保險比划算。

不建議：沒有衍生性商品經驗者不要為了「避險」而開新工具。

普遍建議 — 不論哪種情境

1. 2026/05/10~05/15（Q1 季報前後）絕不重倉進場／加碼。
2. 每週檢查三重頂部（380 / 389.5 / 400 三個關鍵位）是否失守。
3. 關注銥金屬現貨價（影響毛利率直接 ±5%）。

Top 3 盲點洞察

洞察 1：「韓國 45% 邊界」被市場完全忽略。報告公告時皆強調「跨國複製 BOO 成功」，但 45% 持股意味衛司特對韓國業務的實質控制力有限；若當地合資方要求增資或退出，衛司特只能被動。

洞察 2：研發費用僅 1.63% vs 產業平均 3~5%。這不是「護城河深」而是「護城河淺到只靠 De Nora 技術代理」。若 De Nora 撤出或被收購，衛司特技術獨立性將暴露。

洞察 3：接班不等於能力。林浚丞清大化工博士+台積電背景是亮點，但「三合一接班」（總經理+董事長+控股家族）也代表決策集中、缺乏制衡；2025/05 到 2028 的 3 年績效將是真正驗證期。
