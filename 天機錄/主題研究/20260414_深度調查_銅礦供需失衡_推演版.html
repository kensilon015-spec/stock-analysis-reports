重要啟示：路徑C是最佳布局機會——如果發生，回調至$10,000附近是上游礦商的黃金買點。

路徑 D：需求斷崖（黑天鵝 — 10%）

觸發條件：全球經濟衰退 + 中國房地產/地方債危機 + EV補貼全面取消 + AI泡沫破裂

後果：銅價跌至 $8,000-9,000/噸。礦商股跌30-50%。擴產計畫延後。但品位下降+新礦枯竭的結構問題不變——延後缺口但不解決。2032年後缺口可能更大（因投資斷裂）。

對策：停損執行。等經濟觸底後重新布局。

三、蝴蝶效應推衍

3.1 弱訊號 → 連鎖反應

弱訊號一：TC/RC歸零 → 冶煉商退出潮 → 精煉銅也短缺

初始訊號：2026年TC/RC基準 $0/噸（已發生）

一階：無自有礦源的獨立冶煉商開始虧損 → 銀行收緊信貸 → 部分關閉

二階：全球冶煉產能收縮 5-10% → 精煉銅也從過剩轉缺口 → 2027年雙重短缺（精礦+精煉銅）

三階：礦商被迫自建冶煉（IVN已做）→ 產業鏈垂直整合加速 → 獨立冶煉商進一步邊緣化

機率：45%。已有明確跡象（中國減產10%、日本減產、Glencore Mount Isa紓困）。

弱訊號二：Section 232關稅 → 雙軌銅市 → 全球供應鏈重組

初始訊號：2026年4月6日 50%銅製品關稅生效

一階：美國國內銅溢價飆升 → 美國製造業銅成本+50% → 電網更新/AI數據中心建設成本大增

二階：美國國內銅礦開發加速（政策支持+經濟可行性提升）→ 但17年時滯意味短期無解

三階：非美國地區銅價可能暫時受壓 → LME/COMEX價差成為新交易主題 → 套利+實體銅跨區流動

機率：60%（已部分實現，6月30日報告為下一催化劑）

弱訊號三：品位懸崖 → 成本曲線陡峭化 → 高成本礦商出局

初始訊號：品位從1.2%降至0.65%（已發生，持續中）

一階：每噸銅的開採成本持續上升（能源+水+人力）→ 成本曲線右移

二階：低品位高成本礦商（C3成本>$8,000/噸）的利潤壓縮 → 減產或關閉

三階：供給更集中於少數低成本大型礦商（BHP/FCX/SCCO/紫金/IVN）→ 寡占格局強化 → 定價權集中

機率：70%（結構性趨勢，幾乎確定發生，分歧僅在速度）
