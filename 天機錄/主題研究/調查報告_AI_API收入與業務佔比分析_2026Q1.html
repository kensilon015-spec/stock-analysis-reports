核心發現摘要 三家公司的 AI API 收入在 2025-2026 年間呈現爆發式成長。Anthropic 以 API 為核心（佔營收 70-75%），OpenAI 以消費端訂閱為主（API 僅佔約 15-20%），Google 則將 AI API 深度整合進雲端生態系。截至 2026 年初，Anthropic 在企業 LLM 支出中已超越 OpenAI 拿下 40% 市佔。

詳細分析
API 收入快速成長中 Sam Altman 於 2026 年 1 月 23 日表示，僅該月 API 業務就產生超過 $10 億 ARR。但相對於整體 $200 億 ARR，API 仍非主要收入來源。ChatGPT Plus（$20/月）和 ChatGPT Pro（$200/月）的訂閱收入在 2024 年貢獻了約 $55 億，佔比約 75%。企業端有超過 300 萬付費商業用戶。

關鍵特徵：OpenAI 的商業模式以 C 端（消費者）為重心，API 是開發者生態的基石但營收佔比相對低。這反映了 ChatGPT 的消費級品牌效應極強。

API 為核心商業模式 Anthropic 是三家中最「API 優先」的公司。截至 2025 年 10 月，擁有超過 30 萬企業客戶，貢獻約 80% 的營收。Claude Code 於 2026 年 2 月已達 $25 億 ARR，自年初以來翻倍成長。ARR 從 2024 年底的 $10 億，飛速成長至 2026 年 3 月的 $190 億。

關鍵特徵：Anthropic 透過 Amazon Bedrock 與 Google Cloud 的合作夥伴通路取得大量企業客戶，API 即其核心變現引擎。消費端的 Claude Pro/Max 訂閱收入佔比較低。

詳細分析
AI 加速雲端成長 Google Cloud 2025 Q4 營收 $177 億（年增 48%），AI 是主要推動力。近 75% 的 Google Cloud 客戶已使用 Google 的 AI 服務，超過 12 萬家企業使用 Gemini。Gemini API 處理量達每分鐘 100 億 tokens。但 Alphabet 整體年營收 $4,028 億中，AI API 的直接營收佔比極低，主要是做為雲端服務的加值項目。

關鍵特徵：Google 的策略不是靠 AI API 直接賺錢，而是用 AI 強化整個生態系（Search、Cloud、廣告、YouTube），間接提升各業務線營收。AI API 是「戰略性武器」而非獨立營利中心。

價格降幅驚人 2025 至 2026 年間，主要 LLM API 價格平均下降約 80%。GPT-5.2 的定價僅為 GPT-4 上市時的一小部分。這對開發者是好事，但也意味著 API 供應商必須靠「量」來彌補「價」。

目前對產業的影響

企業採用率爆發 Gartner 預測至 2026 年，超過 80% 的企業將使用過生成式 AI API 或部署 AI 應用程式，2023 年這個數字不到 5%。
開發者生態翻轉 84% 的開發者表示正在使用或計畫使用 AI 工具，51% 每天使用。開發者平均每週節省 3.6 小時。
三大影響面向

1. 多模型策略成為主流：企業不再綁定單一供應商，而是根據任務特性選用不同模型。程式開發用 Claude、大量文件處理用 Gemini Flash、對話應用用 GPT。

2. AI Agent 崛起：2026 年企業應用已從「提供數位工具給員工」進化為「管理 AI Agent 數位勞動力」。MCP（Model Context Protocol）的普及使得跨 Agent 通訊成為現實。

3. 虧損換增長模式面臨考驗：OpenAI 預計到 2028 年累計虧損 $740 億，Anthropic 則預計 2027 年就能實現正向自由現金流。商業模式的可持續性差異日益明顯。

五大衍變趨勢

趨勢 1：API 定價持續崩跌，「免費增值」模式浮現

API 價格每 12-18 個月降低 50-80%。Google 已在部分模型提供免費額度，未來可能出現「基礎 API 免費、進階功能收費」的模式，類似雲端計算的發展路徑。這對小型 AI 新創是好消息，但也會壓縮純 API 供應商的利潤空間。

趨勢 2：Anthropic vs OpenAI 的盈利模式分歧加劇

Anthropic（預計 2027 正現金流）走的是「高效 B2B」路線；OpenAI（預計 2030 才能獲利）走「燒錢換生態」路線。若 OpenAI IPO 順利，資金不是問題；但若市場降溫，其龐大虧損將成為巨大壓力。Anthropic 的財務紀律可能成為長期優勢。

趨勢 3：Google 的「圍棋戰略」— 用 AI 加固護城河

Alphabet 2026 年資本支出計畫 $1,750-1,850 億，幾乎是 2025 年的兩倍。Google 不靠 AI API 本身賺錢，而是用 AI 強化搜尋廣告（佔營收 75%）和 Cloud 黏著度。這讓 Google 有能力持續低價甚至虧損補貼 API，對純 API 公司形成定價壓力。

趨勢 4：AI Agent 經濟取代單純 API 呼叫

Claude Code 的 $25 億 ARR 已經證明：未來的贏家不是賣最便宜 token 的公司，而是能提供「完整 Agent 解決方案」的公司。API 只是基礎設施，Agent 才是價值載體。Forrester 預測 2026 年 AI Agent 將根本改變企業軟體市場。

趨勢 5：營收分潤與合作夥伴生態

Anthropic 在 2027 年可能需支付高達 $64 億給 Amazon、Google、Microsoft 等合作夥伴。這種「通路分潤」模式意味著 AI 模型公司不只是技術競爭，還要處理複雜的商業聯盟關係。誰能建立最強的生態系聯盟，誰就能在企業市場勝出。
