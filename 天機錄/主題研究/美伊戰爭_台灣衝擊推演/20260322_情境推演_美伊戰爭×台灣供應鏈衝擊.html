2/28 — D-Day：史詩之怒行動
美以聯軍 20:38 UTC 發動「Operation Epic Fury」，12 小時內執行近 900 次空襲。目標涵蓋德黑蘭國防部、情報部、原子能總部、最高領袖官邸、司法大樓、總統府。
最高領袖哈梅內伊及多名高官遭擊殺。
3/1 — 伊朗確認哈梅內伊死亡
最高國家安全委員會確認死訊。伊朗以飛彈和無人機反擊以色列、美軍基地及海灣盟國（UAE、卡達、巴林）。臨時領導委員會成立。
3/2 — 荷姆茲海峽威脅
IRGC 透過 VHF 無線電宣布海峽「關閉」，威脅攻擊任何通過的船隻。油輪 Skylight 號遭擊中，2 名印度船員喪生。
3/3 — 核設施與指揮中心遭毀
美以聯軍摧毀最高國安會總部、專家鑑定委員會大樓，以及一處代號「Min Zadai」的地下核武設施。
3/4 — 海峽正式封鎖
伊朗軍隊宣布荷姆茲海峽關閉，開始系統性攻擊商船。海峽交通量暴跌 70%，超過 150 艘船滯留海峽外。
3/8 — 布蘭特原油突破 $100
Brent 原油四年來首次突破每桶 100 美元，峰值觸及 $126。卡達宣布 LNG 出口不可抗力。
3/9 — 穆傑塔巴·哈梅內伊繼位
專家會議選出 56 歲的穆傑塔巴·哈梅內伊（前最高領袖之子）為新任最高領袖，但至今未公開露面，美方稱其可能受傷。
3/10 — 全球供應中斷 10M 桶/日
科威特、伊拉克、沙烏地、UAE 合計減產 670 萬桶/日；至 3/12 擴大至超過 1,000 萬桶/日。
3/11 — 沙烏地啟用東西管線全產能
Petroline 管線（Abqaiq→Yanbu）轉為全產能 700 萬桶/日運行，但 Yanbu 港實際裝載能力僅約 300-400 萬桶/日。
3/16 — 海上無人機擴大攻擊
伊朗海上無人機攻擊科威特穆巴拉克港附近油輪 Sonangol Namibe 號（距海峽 800+ 公里），衝突範圍大幅擴大。
3/20 — 卡達 LNG 設施遭襲
伊朗飛彈攻擊卡達 Ras Laffan LNG 終端（報復以色列攻擊南帕斯氣田），全球 LNG 市場劇震。
3/21 — 川普 48 小時最後通牒
川普威脅若伊朗不在 48 小時內重開荷姆茲海峽，將攻擊伊朗發電廠。美以再次攻擊 Natanz 核設施（第四次）。
3/22 — 今日
布蘭特原油 $107.4/桶。新最高領袖下落不明。衝突持續升級中。眾院以 219:212 否決戰爭權力決議案。

電力供應危機 — 深度分析

台灣發電結構中，天然氣發電佔比約 40%。若 LNG 供應中斷持續：

庫存天數狀態可能措施
11 天（法定最低） 緊急啟動燃油/燃煤替代發電、大型用電戶限電
7 天 極度緊急全面限電輪流供電、工業用電管制
3 天 危機核電廠延役討論、緊急進口非中東 LNG
台積電風險 台積電晶圓廠需要極穩定電力，任何電壓波動都可能造成良率損失。限電情境下，台灣半導體產能可能被迫減產，進而影響全球晶片供應。這是一條從中東延伸到全球科技產業的關鍵連鎖。
七、蝴蝶效應 — 多路徑級聯分析
一場中東戰爭如何在 12 條路徑上衝擊台灣

路徑 1：能源斷供 → 電力危機 → 半導體減產 → 全球科技震盪

荷姆茲封鎖 → 卡達 LNG 停供 → 台灣 LNG 庫存 11 天告急 → 天然氣發電減量 → 台電限電/輪流供電 → 台積電/聯電減產 → 全球晶片供應惡化 → 汽車/手機/AI 伺服器延遲交貨

衝擊時間：2-4 週內浮現 | 嚴重度： 極高

此為台灣最獨特且最嚴重的蝴蝶效應路徑，因為台灣半導體佔全球先進製程 90%+，限電即等於全球科技斷鏈。

路徑 2：油價飆升 → 石化成本 → 通膨 → 央行升息 → 房市/股市承壓

油價 $70→$126 → 石化原料漲 40-80% → 塑膠/紡織/包材漲價 → 台灣 CPI 上升 2-3% → 央行被迫升息 → 房貸利率上升 → 房市降溫/股市修正

衝擊時間：1-3 個月 | 嚴重度： 中高

路徑 3：美軍兵力分散 → 台海威懾弱化 → 中國窗口期

美軍投入中東作戰 → 航母/驅逐艦調離印太 → 西太平洋軍力真空 → 中國感知「戰略窗口」 → 台海軍事壓力可能升級

衝擊時間：中期（3-12 個月）| 嚴重度： 極高（潛在性）

歷史先例 2003 年伊拉克戰爭期間，中國軍方內部出現「利用美軍分身乏術推進統一議程」的討論。2026 年的規模更大、持續更久，中國可能進行更大膽的軍事測試。

路徑 4：中國石油恐慌 → 搶購/囤積 → 亞洲能源競爭加劇

中國是伊朗最大石油買家 → 伊朗原油出口中斷 → 中國緊急搶購替代油源 → 亞洲現貨油價溢價擴大 → 台灣/日本/韓國採購成本暴增

衝擊時間：即時 | 嚴重度： 中高

中國每日從伊朗進口約 150 萬桶石油，中斷後將在全球現貨市場與台灣競爭有限的非中東油源。

路徑 5：運費飆升 → 全球供應鏈成本 → 「再通膨」

海峽封鎖 + 紅海胡塞威脅 → 好望角繞行（+15-20天） → 運費翻倍 + 戰爭險 100 倍 → 全球貿易成本上升 → 進口消費品漲價 → 全球央行緊縮困境

衝擊時間：1-3 個月 | 嚴重度： 中

路徑 6：糧食安全 → 海灣國家 80% 糧食中斷 → 難民潮

海峽封鎖 → GCC 國家 80% 糧食進口中斷 → 食品價格飆漲 40-120% → 海灣國家社會動盪 → 南亞/中東難民潮 → 全球人道危機

衝擊時間：2-6 週 | 嚴重度： 極高（區域性）

GCC 國家超過 80% 的熱量攝取依賴海運進口。3 月中旬已有 70% 食品進口遭中斷，零售商被迫空運基本物資。

路徑 7：新台幣貶值 → 進口成本 → 外資撤離

全球避險情緒 → 資金湧入美元/黃金 → 新興市場貨幣承壓 → 新台幣貶值 → 進口能源成本更高 → 雙重打擊（量減+價增）

衝擊時間：即時 | 嚴重度： 中

參考菲律賓披索已跌至 60.1 PHP/USD 的歷史新低。台灣雖有較強外匯存底，但大量外資持有台股，恐慌性撤離仍是風險。

路徑 8-12：其他連鎖效應

路徑 8：核擴散風險 → 中東軍備競賽

伊朗核設施被攻擊但 IAEA 表示無法完全消除核計劃 → 殘餘核知識擴散風險 → 沙烏地/土耳其/埃及加速核計劃 → 中東核擴散 → 全球安全格局改變

路徑 9：俄羅斯漁翁得利

美國注意力轉向中東 → 烏克蘭戰場壓力減輕 → 俄羅斯石油成為替代供應（價格溢價）→ 制裁效果弱化 → 俄羅斯戰略地位提升 → 全球權力格局重塑

路徑 10：氦氣/鋁/肥料供應鏈

卡達是全球第二大氦氣生產國 → LNG 設施停產連帶氦氣停供 → 半導體/醫療/科研用氦氣短缺 → 又一條衝擊台灣半導體的隱形路徑

路徑 11：電價上漲 → 民生壓力 → 社會情緒

LNG 漲價 60% → 台電發電成本暴增 → 電價調漲壓力 → 製造業成本上升 → 物價全面上漲 → 民眾購買力下降 → 消費萎縮 → 經濟放緩

路徑 12：保險/再保險危機

戰爭險費率從 0.05% 飆至 5%+ → 部分船東拒絕進入高風險水域 → 即使海峽部分重開，保費成本也永久墊高 → 長期貿易成本結構性上升

核心結論

台灣面臨三重風險疊加 1. 能源安全：LNG 僅 11 天庫存是最急迫的威脅，天然氣發電佔 40%，斷供將直接導致限電。
2. 經濟衝擊：油價飆升 + 運費翻倍 + 原物料漲價，CPI 上升 2-3% 以上，停滯性通膨風險。
3. 國防窗口：美軍分身乏術，中國可能評估這是對台施壓的戰略機會窗口。

最壞的蝴蝶效應路徑：荷姆茲封鎖 → LNG 斷供 → 台灣限電 → 台積電減產 → 全球晶片危機 → 科技股崩跌 → 全球經濟衰退。這條路徑的每一環都已開始出現壓力。
時間是關鍵變數 台灣石油儲備 120 天提供了緩衝，但 LNG 僅 11 天的庫存意味著天然氣供應是最先也最嚴重的瓶頸。在情境 B（持續衝突 3-6 個月，機率 40-45%）下，台灣將在 2-4 週內面臨 LNG 供應危機的臨界點。
