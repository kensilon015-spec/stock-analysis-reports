8. 半導體供應鏈衝擊
氦氣硬停頓倒數 + KOSPI 史上最大崩跌
8A. 氦氣危機
4 月底為「硬停頓」節點 多數晶圓廠特殊氣體庫存僅 4-8 週緩衝。三星/SK 海力士稱庫存約 6 個月。氦氣在 EUV 光刻（冷卻）與蝕刻（散熱）中不可替代。半導體產業佔全球氦消耗 10-12%。卡達停產切斷全球 30% 供應（月缺口 520 萬立方公尺）。現貨價已翻倍，若持續 60-90 天恐漲 4 倍。替代來源（美國/俄羅斯 Amur 2/阿爾及利亞）僅能覆蓋約 50%。中國國產超高純度氦氣僅滿足 5% 國內需求。
8B. 溴素風險

全球 2/3 溴素集中以色列+約旦死海（ICL 佔全球 1/3）。韓國 90% 溴素來自以色列。溴素用於半導體蝕刻（HBr）+PCB 阻燃劑。以色列 Tower Semiconductor 出貨受阻。3/19 雙方互攻能源設施，死海產區面臨直接戰場風險。

8C. 韓國半導體崩盤

KOSPI 3/4 暴跌 12.06%（超越 911 的 12.02%），前一日已跌 7.2%。三星 -9.88%、SK 海力士 -11.5%。合計蒸發超過 $2,000 億市值。KOSPI 三度觸發 Sidecar。融資槓桿強制平倉加劇崩跌。韓國 70% 原油來自中東、65% 氦氣來自卡達、90% 溴素來自以色列 — 能源與材料「三重斷供」風險。政府已將 14 種半導體材料列為高度脆弱監控對象。

8D. 台積電與台灣半導體

台積電官方聲明：目前營運正常，庫存與供應可管理。但市場未買單，股價因氦氣消息承壓。台積電獨佔台灣 ~10% 電力，限電等於全球 AI 晶片斷供。若材料短缺加劇，分析師警告可能優先生產高毛利 AI 晶片，犧牲汽車/消費電子（重演 2021 年車用晶片荒）。

LNG 替代方案：中油已將 3-4 月卡達 22 船缺口全部補齊（美國/澳洲替代）。LNG 來源已分散至 14 國（卡達 49%→25%）。目標 2029 年美國佔 25%。
核電重啟：3/21 賴清德宣布核二/核三具重啟條件，3 月底送核安會。但最快 2028 年才能重啟。
備用方案：經濟部長稱 3 月供氣無虞，4 月後需因應。全面燃煤為最後手段。
發電結構：天然氣 ~50%、燃煤 ~30%、再生能源 10-15%、核能 <3%。

11. 系統動力學
反饋迴路與級聯風險鏈
11A. 正反饋迴路（自我增強）
迴路 R1：油價-通膨螺旋
油價飆升 → 全球通膨上升 → 央行被迫緊縮 → 經濟衰退風險 → 需求下降 → 但供應中斷更大 → 油價繼續飆升
迴路 R2：保險-封鎖螺旋
商船遭攻擊 → 保險公司退場 → 保費飆漲 → 更多船拒絕通行 → 海峽事實封鎖 → 供應更緊張 → 油價更高 → 更多攻擊利益
迴路 R3：升級螺旋
以色列攻擊南帕爾斯 → 伊朗報復攻擊卡達+以色列煉油廠 → 美國威脅攻擊發電廠 → 伊朗威脅攻擊美以能源設施 → 持續升級
11B. 負反饋迴路（自我修正）
迴路 B1：經濟壓力制動
油價飆升 → 美國國內反對升高（47% 反對、中期選舉壓力）→ 共和黨內部分裂 → 政治壓力迫使收斂
迴路 B2：伊朗能力耗竭
持續發射飛彈/無人機 → 庫存耗盡（飛彈發射率已降 90%、無人機降 83%）→ 打擊力下降 → 被迫接受談判
11C. 級聯風險鏈

鏈 1：能源 → 半導體 → 全球 AI
卡達停產 → 氦氣斷供 → EUV 無法運作 → 先進晶片停產 → 全球 AI 基建延遲 → 超大規模客戶調降 CapEx

鏈 2：海峽封鎖 → 台灣限電 → 全球晶片
LNG 斷供 → 11 天存量耗盡 → 天然氣發電停擺（佔 50%）→ 台灣限電 → 台積電產能受限 → 全球晶片短缺

鏈 3：油價 → 化肥 → 糧食 → 社會穩定
油價翻倍 → 化肥暴漲 32% → 北半球春耕受阻 → 農產品成本上升 → 台灣糧價通膨（70% 進口）

鏈 4：保險退場 → 航運成本 → 全球貿易
戰爭險取消 → 實質商業封鎖 → 運費附加費 $2,000-4,000/櫃 → 全球供應鏈成本全面上升
