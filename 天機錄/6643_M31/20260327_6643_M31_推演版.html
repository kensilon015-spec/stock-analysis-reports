8c. 影響鏈分析（推衍方法一）

從關鍵事件出發，推演一階、二階、三階影響，並評估跨鏈交互效應。

影響鏈 1：3nm/2nm 量產放大 → 權利金爆發

起始事件：台積電 3nm 量產規模持續放大，2nm 進入試產

→ 一階影響：M31 的 3nm/2nm IP 權利金按出貨量計費，被動收入隨量產規模自然成長
→ 二階影響：權利金佔營收比從 15% → 20% → 25%，營收結構從「一次性授權」轉向「經常性收入」
→ 三階影響：資本市場將 M31 重新歸類為「recurring revenue」模式，PE 倍數從目前 40-60x 重估至 80-100x
機率：65% 影響：EPS +2~4 元/年 時間窗口：2026H2 - 2027

影響鏈 2：中國出口管制升級

起始事件：美國將半導體 IP 授權正式納入出口管制清單

→ 一階影響：中國客戶（佔營收 43.3%）的新授權案全面中斷
→ 二階影響：已簽約的權利金可能繼續收取（存量合約），但新授權凍結導致營收成長引擎熄火
→ 三階影響：M31 被迫加速轉向日本、歐洲、印度等非中國市場，短期 1-2 年出現營收缺口
機率：20% 影響：營收 -30~40% 時間窗口：不確定（政策驅動）

影響鏈 3：EDA 費用持續膨脹

起始事件：Synopsys/Cadence EDA 長約到期後再次大幅提價

→ 一階影響：研發費用率從目前 68% 進一步升至 80% 以上
→ 二階影響：營益率被壓縮至 0% 附近，公司陷入「有營收無利潤」的泥淖
→ 三階影響：為控制成本被迫縮減先進製程（2nm/1.4nm）研發投入 → 競爭力下降 → 客戶轉向 Synopsys/Cadence 自有 IP
機率：30% 影響：EPS -3~5 元 時間窗口：長約到期時
跨鏈交互效應（最壞情境） 若影響鏈 2（出口管制升級）+ 影響鏈 3（EDA 費用膨脹）同時發生 → 營收萎縮 + 成本膨脹 = 雙面夾擊 → 對應 H4「結構性衰退」情境（機率 11%）。在此情境下，M31 可能面臨生存危機，股價恐跌至 150-250 元。
跨鏈交互效應（最好情境） 若影響鏈 1（權利金爆發）順利推進 + 影響鏈 2 未發生 + 影響鏈 3 透過 AI-EDA 工具緩解 → 對應 H1「強勢反轉」情境（機率 22%）。在此情境下，2027 年 EPS 可達 10-16 元，股價有機會回到 600-900 元。
8c 整合結論 三條影響鏈的機率加權淨效應：影響鏈 1（正面 65%）力道最強但需時間兌現；影響鏈 2（負面 20%）是黑天鵝型風險，機率不高但衝擊致命；影響鏈 3（負面 30%）是慢性病，持續侵蝕但不會突然致命。推論：三條鏈中，影響鏈 1 是唯一能讓 M31 脫離平庸的正面催化劑，但它的兌現需要 2-4 季。在等待期間，影響鏈 2 和 3 隨時可能惡化。因此，時間是 M31 投資者的朋友（權利金累積），也是敵人（風險累積）。

8g. 蝴蝶效應推衍（推衍方法八）

掃描主流分析師未關注的弱訊號，推演可能的蝴蝶效應鏈。

弱訊號掃描

A 類 數據異常：2024/11 營收暴跌至 6,937 萬（MoM -52%），但 12 月即反彈至 1.28 億。表面看像客戶流失，實際可能是案件認列時間差（IP 授權金認列依合約里程碑，不是每月平均）。推衍意義：M31 的月營收波動度極大，不宜用單月數據判斷趨勢，至少需要看季度。
B 類 敘事轉變：投資論壇出現「跌落神壇」「法說會又變法會」等極度悲觀敘事。歷史經驗顯示，當散戶共識達到極端悲觀時，往往是反轉的前兆（但不是立即反轉，通常需要 1-2 季基本面驗證）。推衍意義：市場情緒已充分反映壞消息，若出現正面催化劑，反彈力道可能超預期。
C 類 結構變化：Chiplet 架構興起 — 一顆傳統 SoC 被拆分成多個 chiplet，每個 chiplet 之間需要高速介面 IP（如 UCIe、PCIe、HBM controller）進行互連。這意味著每顆 SoC 的 IP 需求量可能從 1 組變成 3-5 組。M31 的高速介面 IP 正好處於這個風口。推衍意義：這是一個被市場嚴重低估的結構性利多。
D 類 關係變化：摩爾線程（中國 GPU 新創，2022 年與 M31 簽約）若被美國制裁，M31 損失一個客戶但免除法律風險。反之，若摩爾線程成功量產，權利金收入將顯著增加。推衍意義：中國 AI 晶片客戶是雙面刃。
E 類 時間窗口：台積電 2nm 製程 2026 年開始試產，M31 已簽約成為 2nm IP 供應商。若量產順利，2027-2028 年的權利金將進入大爆發期。推衍意義：這是 M31 最大的長期選擇權。

觸發機率：25% 影響量級：大（雙面刃） 時間：2026-2027
洞察：短期甜蜜、長期致命。中國 AI 晶片自主化對 M31 是「毒蘋果」— 吃了營收好看，但依賴度越高、被管制時的衝擊越大。

→ EPS 在營收不變的情況下也能倍增
觸發機率：20% 影響量級：中大 時間：2027-2028
洞察：EDA 費用是 M31 最大的利潤侵蝕因子。如果 AI-EDA 能有效降低成本，等於同時解決了利潤問題和研發效率問題。但此路徑需要 M31 管理層有前瞻性佈局。
8g 整合結論 弱訊號掃描發現一個被嚴重低估的結構性利多：Chiplet 架構的 IP 倍增效應。這不是「可能會發生」而是「正在發生」— Intel、AMD、NVIDIA 都已採用 Chiplet，問題只在普及速度。若 Chiplet 在 2027-2028 全面普及，M31 的高速介面 IP 需求可能倍增，這是目前市場定價中完全沒有反映的選擇權。

但同時，中國 AI 晶片自主化是一個「甜蜜的陷阱」— 短期營收好看但長期風險累積。推論：M31 的長期價值取決於兩個變數的賽跑：（1）Chiplet + 2nm 權利金能否在（2）中國風險爆發前兌現。如果（1）先到 → 股價 700+；如果（2）先到 → 股價 250-。
8h. 貓膩推衍
