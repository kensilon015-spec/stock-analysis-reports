五條影響鏈（台股視角）

鏈 1：先進製程護航 — 台積電獨佔效應
台美 ART 協定：半導體 232 最惠待遇 ↓ 台積電 3nm/2nm 完全豁免 ├→ 蘋果 M5/A20 繼續下單 → 台積電營收穩固 ├→ NVIDIA Blackwell 獨家代工 → 產能滿載 ├→ AMD MI400 代工 → ASP 持續上升 └→ 全球無替代者 → 議價權史上最強 ↓ 連帶受惠： ├→ 日月光/矽品 CoWoS 先進封裝（供不應求） ├→ 欣興/南電 ABF 載板（2nm 用量增） └→ 創意/世芯 ASIC 設計（搭台積電順風車）

時間軸：已生效。2026 全年台積電營收預估 +20% YoY

信心度：高（ART 已簽署，客戶長約已鎖定）

鏈 2：成熟製程轉單 — 中芯受限的紅利
中芯國際設備取得受限 + 美國客戶迴避中國代工 ↓ 28nm 以上成熟製程訂單外溢 ├→ 聯電（28/22nm）：車用 MCU + 電源管理 IC 轉單 │ 產能利用率可能從 65% → 75%（+10%） ├→ 世界先進（0.15μm）：電源管理/驅動 IC │ 車用客戶（TI/NXP）降低中國依賴 └→ 力積電：NOR Flash + 利基型 DRAM 但自身財務壓力大，受惠程度有限 ↓ 風險：中芯 2027 年用現有 DUV 設備狂擴產 → 成熟製程全球供過於求 → 價格戰 → 聯電/世界先進 2027~2028 面臨價格壓力

時間軸：轉單 2026 Q2~Q3 顯現，供過於求風險 2027 H2

信心度：中高（轉單趨勢明確，但中芯擴產速度是變數）

鏈 3：AI 伺服器代工 — 關稅成本誰吸收
台灣組裝出口美國課 15% 關稅 ↓ AI 伺服器（含 NVIDIA GPU）成本上升 ├→ 廣達：Meta/Microsoft 主要 ODM │ 毛利率 4.5% → 可能壓至 3~3.5%（關稅吃掉 1%） │ 已啟動墨西哥產線（2026 Q3 量產） │ ├→ 鴻海：Apple + NVIDIA + Amazon │ 規模最大但毛利最薄（~3%） │ 印度 iPhone 組裝佔比目標 25% │ 墨西哥 AI 伺服器組裝加速 │ ├→ 英業達/緯創：Dell/HPE 訂單 │ 墨西哥產能較少，短期壓力最大 │ └→ 誰吸收關稅？ 品牌客戶（NVIDIA/Apple）議價力強 → 大部分轉嫁 ODM ODM 唯一出路：搬去墨西哥（0%）或美國（抵免）

時間軸：關稅已生效，毛利壓縮 Q1 財報即可見。墨西哥轉移 2026 H2~2027

信心度：高（15% 稅率已確定，ODM 毛利結構公開）

鏈 4：稀土反制 — 化合物半導體受壓
中國限制鎵/鍺/銻出口（已執行中） ↓ ├→ 穩懋（GaAs/GaN 代工） │ 鎵原料成本漲 300% → 毛利嚴重壓縮 │ 5G PA/濾波器客戶可能轉向 Si 替代方案 │ ├→ 環球晶（SiC 基板） │ 碳化矽不直接受鎵限制，但供應鏈情緒連動 │ 電動車 SiC 需求仍強 → 影響中性 │ └→ 被動元件（國巨/華新科） 稀土永磁材料 → MLCC 不受影響 但鈷/鎳若後續被限制 → 電池產業鏈受衝擊 目前尚未發生，列觀察

時間軸：鎵/鍺已限制中，擴大到其他稀土隨時可能

信心度：中（鎵已確認，其他材料是否跟進不確定）

鏈 5：供應鏈大遷移 — 台灣定位重塑
全球關稅壁壘 → 「在哪裡製造」比「誰製造」更重要 ↓ 台灣留下什麼： ├→ 先進製程代工（台積電 2nm 留台灣） ← 不可替代 ├→ 先進封裝（CoWoS/InFO） ← 不可替代 ├→ IC 設計（聯發科/瑞昱/聯詠） ← 輕資產不受關稅 └→ 高階檢測設備（精測/牧德） ← 利基型 台灣外移什麼： ├→ 系統組裝（鴻海/廣達 → 墨西哥/印度） ← 已在進行 ├→ 被動元件產能（國巨 → 泰國/越南） ← 加速中 ├→ PCB 中低階（健鼎 → 泰國） ← 趨勢明確 └→ 面板模組（群創 → 越南） ← 已執行 台股結構性變化： ├→ 「留台」公司 → 估值溢價提升（護城河+不可替代） ├→ 「外移」公司 → 短期成本壓力，中期受惠（避關稅） └→ 「卡中間」公司 → 最危險（想走走不了，留下又被課稅）

時間軸：3~5 年結構性轉變，已不可逆

信心度：高（鴻海/廣達/國巨已公布遷移計畫）
