8c. 影響鏈分析 — 三大事件的骨牌效應
每個事件不是孤立的，它們會觸發連鎖反應。

影響鏈推演

三階（結構性影響）：若 RODA 後續合約持續自行履約（有雙線產能支撐），茂訊系統事業部營收將出現結構性下滑，不是週期波動而是永久性流失。

白話說：以前是「MilDef 吃不下的茂訊來接」，現在 MilDef 蓋了更大的廚房，不需要外包了。

關鍵事件 2：MilDef 雙線擴產（Rosersberg 4x + Helsingborg）

影響鏈推演

三階（中期脫離）：2027-2028 年，RODA 品牌訂單可能完全轉回歐洲生產。茂訊從「策略夥伴」降格為「備援產能」，甚至連備援都不需要。

白話說：MilDef 蓋的不是一條線，是兩條。這不像「怕不夠用所以多蓋」，更像「準備把外包全收回來」。

影響鏈推演

8g. 蝴蝶效應推衍 — 弱訊號收斂偵測
天機閣的核心能力：多個微小異常同時指向同方向 → 推衍出還沒人看到的趨勢

弱訊號分類清單

A 類 — 跨產業訊號
MilDef 收購 RODA + 自建雙線 + 年報不提茂訊 = 垂直整合策略執行
觀察：這不是三個獨立事件，是一個策略的三個步驟——收購供應鏈（RODA）→ 擴充自有產能（Rosersberg + Helsingborg）→ 淡化外部依賴（年報零提）。
若放大：這是典型的「OEM 被品牌商甩掉」劇本。歷史上最經典的案例是蘋果逐步收回 GPU 設計（Imagination Technologies 股價腰斬 70%）。MilDef 正在對茂訊做同樣的事。
FIAS：推論
B 類 — 時間序列異常
營業利益率連 4 季下滑：26% → 18% → 12% → 7%
觀察：這不是隨機波動，是穩定的線性下滑（每季約 -6 個百分點）。如果是季節性因素，應該會有回彈的季度。連續 4 季不回彈 = 結構性問題。
可能原因：RODA 議價能力增強（被 MilDef 收購後可以壓低代工價）→ 茂訊代工毛利被壓縮 → 即使不斷單，每筆單的獲利也在縮水。
若放大：如果 2026 Q1 OPM 跌破 5%，代表系統事業部已接近損益兩平，即使有單也不賺錢。這比「斷單」更糟——因為市場可能要到 OPM 轉負才會反應。
FIAS：推論（基於 4 季財報數據的趨勢推理）
C 類 — 反共識（最大矛盾）
管理層「訂單能見度九成」 vs 營收 YoY -23%
觀察：這是茂訊整個案例中最大的一個矛盾。如果訂單能見度真的九成，營收不應該衰退 23%。兩者不可能同時為真。
可能解釋 1：「訂單能見度」是意向性訂單（有詢問但未下單），非確認訂單。管理層用最樂觀的定義。
可能解釋 2：訂單有但延遲出貨，Q2 後會補回來。但這需要 Q2 營收暴增 30%+ 才能抵消。
可能解釋 3：管理層知道 RODA 在減單但不能說（合約保密條款），用「訂單九成」維持市場信心。
若放大：如果 Q1 營收繼續 -20% 以上，解釋 3 的機率將大幅上升，市場信心可能一次性崩潰。
FIAS：假設（三種解釋各有可能，需 Q1 數據驗證）
D 類 — 行為異常
茂訊邊擴產 +50% 邊處分 MilDef 持股（自相矛盾行為）
觀察：如果管理層真的看好 MilDef 的訂單溢出，為什麼要賣掉 MilDef 的股票？擴產 +50% 的資金需求可以理解，但從 12% 砍到 4.99% 代表賣了超過一半的持股。
可能解釋 1：管理層已經知道 MilDef 的策略轉向，提前獲利了結。
可能解釋 2：擴產資金需求大，被迫減持籌資。但茂訊負債比不高，不太像被迫。
若放大：如果持股繼續降至 4.99% 以下（無需公告），代表茂訊可能已經放棄 MilDef 這條線，正在默默撤退。
FIAS：推論
E 類 — 隱藏關聯
MilDef Year-End 零提茂訊但 RODA 提 52 次 → RODA 已完全被視為「自己人」
觀察：RODA 在 MilDef 年報中出現 52 次，涵蓋策略、產品、訂單、未來展望。茂訊出現 0 次。這個對比本身就是最強的信號——在 MilDef 的世界觀中，RODA = 自己人，茂訊 = 不存在。
隱藏邏輯：2018 年 MilDef 收購 RODA 的股權，目的就是將 OEM 能力內化。到 2025 年報，這個內化已經完成——RODA 不再是「被收購的公司」，而是「集團核心子公司」。茂訊在這個過程中被徹底替代。
若放大：下一步 MilDef 可能將 RODA 品牌併入 MilDef 主品牌，進一步消除外界對 RODA 獨立性的認知。茂訊連「RODA 的代工廠」這個角色都可能消失。
FIAS：推論

蝴蝶效應推演鏈 Top 3

推演鏈 #1：歐洲在地化 → RODA 訂單轉回 → 茂訊純通路股化（機率最高）
MilDef 歐洲在地化策略成形 → 北約客戶偏好「歐洲製造」標籤 → RODA 訂單優先在 Rosersberg / Helsingborg 生產 → 茂訊系統事業部 3 年內營收腰斬（從佔比 66% 降至 30%） → 失去成長股光環，市場重估為純通路股 PE 8-10x → EPS 4-5 元 × PE 8-10x = 股價 32-50 元

白話說：歐洲軍工越來越強調在地生產，MilDef 有自己的工廠了，為什麼還要跨半個地球找台灣代工？3 年後茂訊可能變回一家普通的電腦通路商。

推演鏈 #2：OPM 崩潰 = RODA 壓價 → 即使不斷單也不賺錢
RODA 被 MilDef 收購後議價權增強 → 壓低代工價格（你不接有人接） → 茂訊 OPM 從 26% 連季下滑至 7% → 如果 Q1 跌破 5%，代工接近損益兩平 → 市場發現「有單不賺」比「沒單」更糟（因為佔用產能卻不獲利） → 擴產 +50% 的投資回收期拉長，折舊壓力增加

白話說：比斷單更陰險的是壓價。你還有訂單，但每筆都不賺錢。而你已經花大錢擴了產，產能閒置的折舊會把利潤吃光。

推演鏈 #3：管理層言行矛盾 → 敘事崩解 → 估值重估
管理層持續宣稱「訂單九成能見度」 → Q2 營收仍未轉正 → 法人開始質疑管理層信用 → 賣方下修評等 → 散戶恐慌性拋售 → 股價可能在 1-2 個月內快速修正 30-40%

白話說：「狼來了」的故事。如果連續兩季營收打臉管理層的樂觀說法，市場信任一旦崩潰，修正速度會比基本面惡化快很多。
