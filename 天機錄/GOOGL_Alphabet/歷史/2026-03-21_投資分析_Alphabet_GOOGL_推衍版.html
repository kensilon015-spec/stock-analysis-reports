3 影響鏈分析

影響鏈 #1：CapEx 翻倍 → AI 基礎設施擴張
觸發事件：2026 CapEx 預算 $175-185B（較 2025 近乎翻倍）
第一層影響（直接）
資料中心全球擴建 → 折舊費用年增 $150-250B → 短期營業利潤率承壓 → 自由現金流（FCF）可能驟降 90%
第二層影響（中期）
AI 計算容量暴增 → Cloud + Gemini 服務擴大 → 企業採用加速 → Cloud 2027 年可望達 $1,200-1,500 億營收
第三層影響（長期分歧）
成功情境

AI 投資回報兌現 → PE 重估 28-30x → 目標價 $380-420

失敗情境

重蹈 Meta 2022 情境 → PE 壓縮至 18x → 目標價 $240-280

影響鏈 #2：Gemini 平台效應爆發
觸發事件：DAU 翻倍至 25% + API 850 億月請求 + Apple-Siri 整合預期
第一層影響（直接）
開發者生態形成 → 240 萬開發者進駐 → 應用爆發 → 鎖定效應（Lock-in）加深
第二層影響（中期）
企業客戶 12 萬家以上 → Cloud ARPU 提升 → 毛利率改善
第三層影響（長期分歧）
成功情境

搜尋 + Cloud + AI 三引擎並行 → 平台獨佔地位 → 估值重估，PE 可達 30x+

失敗情境

OpenAI / Anthropic 追上 → 回到競爭均衡 → Gemini 淪為多選項之一

影響鏈 #3：AI 搜尋競爭
觸發事件：ChatGPT 883M 月活用戶 + Google 搜尋市佔首次回升 +0.17%
第一層影響（直接）
市佔止跌（+0.17%） → AI Overviews 反擊見效 → 使用者回流訊號
第二層影響（中期）
零點擊率 43% → 廣告模式被迫轉型 → CPC 上升但點擊量下降 → 廣告營收結構性變化
第三層影響（長期分歧）
成功情境

護城河升級為 AI 壟斷 → 搜尋 + AI 雙重鎖定 → 市佔穩守 88%+

失敗情境

市佔滑至 80% 以下（2-3 年內） → 搜尋廣告營收明顯下降

影響鏈 #4：反壟斷裁決
觸發事件：廣告技術案閉庭辯論完成，裁決進入倒數
第一層影響（行為性補救）
短期利空出盡 → 不確定性移除 → 股價反彈
第二層影響（結構性剝離 AdX）
重估 $30-50 億營收損失 → PE 壓縮 2-3x → 短中期負面衝擊
第三層影響（長期分歧）
兩案均行為性補救

最大利空解除 → 估值跳升 → 法律折價消除

AdX 被強制剝離

中期負面但可能釋放隱含價值 → 拆分後各部門獨立估值可能更高
