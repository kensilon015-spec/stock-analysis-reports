收入結構深層發現（第十步新增）
關鍵發現：匯兌價差才是真正的利潤引擎 公開說明書揭露「其他收入淨額」佔營收 50-53%。東聯的營業收入（5.65 億）主要反映手續費，但真正的大頭是匯兌價差收入（買入台幣→賣出東南亞貨幣的匯率差額），估計額外 2-3 億。

推衍含義：
(1) 真實獲利能力比表面營收更強（總收入可能 8-9 億）
(2) 匯兌價差受匯率波動影響 → Q2 高/Q4 低可能是匯率季節性
(3) AI 動態匯率定價系統的價值被低估 — 它不只是「報價工具」，是「利潤最大化引擎」
(4) 競爭壓力不只看手續費，還要看誰的匯率更優惠 → 這是東聯的技術護城河
風險：若台幣急升 10% 對東南亞貨幣，可能造成單季匯兌損失。但自然避險機制（應收應付同幣別）可部分抵消。

8c. 影響鏈分析

從觸發事件推演至最終股價影響，每條鏈標記機率與 FIAS

因果鏈 1：業績超預期 → 配息吸引 → 法人進場

觸發事件 事實：2025 EPS 14.61 超預期 + 配息 10 元（殖利率 5.4%）

一階影響（機率 80%）：存股族/高股息 ETF 基金注意到 → 買盤增加 → 成交量放大

二階影響（機率 60%）：券商因成交量提升開始覆蓋 → 出具研究報告 → 法人啟動評估

三階影響（機率 40%）：投信/外資開始建倉 → PE 從 12x 重定至 15-18x → 股價 220-260+

累積機率：80% x 60% x 40% = 19.2%（三階全部實現）

白話說：配息 10 元是一塊敲門磚。在台灣市場，5% 以上殖利率的高成長股極為稀少，這會吸引存股族注意。一旦成交量起來，券商自然會跟進覆蓋。但從配息公告到法人實際建倉，通常需要 3-6 個月。

因果鏈 2：移工政策加速 → 市場擴大 → 營收加速

觸發事件 事實：2026/01 服務業移工解禁 + 匯款上限 40→50 萬

一階影響（機率 85%）：移工人數從 86.58 萬加速成長至 90 萬+（2026 年底）

二階影響（機率 70%）：新增移工中 20-30% 使用東聯平台 → 活躍用戶突破 28 萬

三階影響（機率 65%）：交易量 +25%+ → 營收 +20-30% → EPS 成長延續

累積機率：85% x 70% x 65% = 38.7%

白話說：移工增加是最確定的催化劑。服務業解禁（旅宿/餐飲）新增的移工多數來自印尼和越南，恰好是東聯 IndoMoney / VietMoney 的強勢市場。關鍵是新增移工的轉化速度。

因果鏈 3：海外布局驗證 → 第二曲線

觸發事件 假設：港/日執照開始貢獻營收（2026 目標 3-5%）

一階影響（機率 35%）：海外營收達標 → 證明模式可複製

二階影響（機率 25%）：市場給予「國際化溢價」→ PE 上調 3-5x

累積機率：35% x 25% = 8.8%

白話說：海外是 2027+ 的故事。2026 年能看到的頂多是「有沒有虧太多」。如果海外營收超過 3%、且不虧損，就是正面信號。但不應將海外成功納入基本情境。

因果鏈 4（逆風）：競爭加劇 → 毛利壓縮

觸發事件 事實：第五家業者 2025/06 核准 + 銀行端可能降費

一階影響（機率 45%）：新進業者用低價搶客 → 部分用戶流失或費率壓力

二階影響（機率 30%）：毛利率從 55% 壓縮至 50-52%

累積機率：45% x 30% = 13.5%

白話說：目前看來新進業者的規模太小，不足以動搖東聯的 40% 市佔。真正的威脅是銀行端——如果銀行把移工匯款手續費降到 200 元以下，會壓縮東聯的費率優勢。但銀行降費的機率低（移工不是銀行的核心客群）。

8c 整合結論 資料：4 條因果鏈涵蓋最可能的正面和負面路徑。
觀察：鏈 2（移工政策）的累積機率最高（38.7%），是最可靠的成長引擎；鏈 1（法人進場）的影響幅度最大但時間不確定；鏈 4（競爭）的機率和影響都相對有限。
推論：核心成長引擎（移工增+滲透率提升）與估值催化劑（法人覆蓋）是兩個獨立的正面因子，不互相依賴。即使法人遲遲不來，只要營收持續成長+配息維持，股價仍有基本面支撐。
結論：整體偏多但非一面倒。合理的預期是「基本面確定向好，但股價表現取決於市場認知速度」。
