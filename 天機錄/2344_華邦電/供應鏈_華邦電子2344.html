影響傳導路徑

正面情境

HBM需求持續排擠傳統DRAM產能

傳導鏈：三星/SK海力士/美光全面轉向HBM → 傳統DDR3/DDR4產能被排擠 → 利基型DRAM結構性缺貨 → 華邦電DRAM漲價+滿載 → 高雄廠擴產效益放大

受影響方：南亞科(2408)同步受益、華東(8110)封測接單增、下游工控/車用客戶面臨漲價

發生機率：高（已在發生中）

高雄廠16nm量產順利

傳導鏈：16nm DRAM量產 → 支援8Gb DDR4/LPDDR4 → 打入更高價值市場 → ASP提升 → 毛利率持續改善

受影響方：華東(8110)新製程封測、ASML/Lam Research設備訂單、下游車用客戶

發生機率：中高（預計2026H2）

CUBE邊緣AI記憶體放量成功

傳導鏈：CUBE量產 → 邊緣AI低功耗記憶體需求被滿足 → 華邦電開闢第二成長曲線 → 台積電先進封裝產能需求增加

受影響方：台積電(2330)先進封裝、邊緣AI晶片設計公司

發生機率：中（2026年為起步年）

NOR Flash AI伺服器需求超預期

傳導鏈：AI伺服器NOR搭載量倍增 → 華邦電NOR Flash產能滿載 → 漲價轉嫁 → 旺宏同步受益

受影響方：旺宏(2337)同步受益、AI伺服器品牌廠面臨零件成本上升

發生機率：中高

負面情境

ASML設備交期延遲

傳導鏈：ASML DUV設備延遲 → 高雄廠擴產時程推遲 → DRAM產能無法如期擴充 → 訂單轉移風險

受影響方：華東(8110)封測量受影響、下游客戶缺貨加劇

發生機率：中

中國記憶體廠殺入利基型市場

傳導鏈：兆易創新等中國廠擴大低階NOR Flash/DRAM產能 → 價格競爭加劇 → 華邦電消費性產品毛利受壓

受影響方：旺宏(2337)低階產品同受影響

發生機率：中
