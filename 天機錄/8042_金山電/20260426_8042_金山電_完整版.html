第一章：公司概覽 — 你在看的是什麼公司？

公司全景 · 商業模式與營收結構 · 製造能力 · 集團架構 · 資本結構 · ESG 治理
一句話定位 金山電（Jamicon）是全球第 3 大鋁電解電容廠（液態 75% / 固態 25%、台灣液態 + 固態雙料第 2），1970 年成立、2004 上櫃、董事長江士勲（總經理為其子江慶信）；總部新北三重、產線佈局台灣 + 廣州金立 + 泰國三廠（2025 Q3 量產）。2025 年靠 AI 伺服器（GB300 / BBU）+ 一貫化製程降本 + 銀銅漲價傳導三引擎，毛利率 5 季加速衝至 23.66%、EPS YoY +48% 至 0.89 元、月營收 2026/02 YoY +50.17%。但結構張力在 PE 62.92x（歷史中位數 60.7 之上）、配息率 233% 隱性收割、新會計師（資誠）2026 Q2 首次查核、廣州金立 CNY 8,600 萬背書（佔股本 27.8%）、研發預算 2024-2025 砍 25-46%——四次信心校準持續遞減（42% → 38% → 35% → 33%），是本案定價討論的核心錨點。

1.1 公司全景新

產業角色與影響力

金山電在 AI 伺服器供電供應鏈中定位為「Tier-2 鋁電解電容核心零件廠」——間接對接 NVIDIA GB300 / Hyperscaler 客戶（透過群創 / 仁寶 / 鴻海 ODM 一級代工廠），上游整合自家鋁箔與電解液製程（一貫化），下游進入伺服器電源（PSU）/ BBU 電池備援單元 / 5G 通信電源等高品質應用。在台灣液態 + 固態鋁電容雙料市場排名第 2（次於立隆電 2472），規模上是立隆電的 1/3，但護城河在於「HITACHI AIC 高壓電解技術授權 + 鋁箔自製成本優勢」。

在被動元件產業扮演三重定位：① 規模上的第二集團主將——日系（Nippon Chemi-con / Nichicon / Rubycon）合計握有全球 60-70% 高階份額、台灣立隆電 2472 為產業老大、金山電與鈺邦 6449 共同構成台灣第二集團；② 雙路徑覆蓋者——少數同時具備液態（容量主流）與固態（高頻 AI / 車用主流）量產能力的廠商，相對純液態廠（如部分中國中小廠）有產品組合彈性；③ 非中國製造優勢——泰國三廠 2025 Q3 量產，使其成為 ODM 客戶在「中國 +1」分散風險下的潛在轉單對象，地緣政治避險紅利明確。

客戶切換成本中等：鋁電容是「規格化 + 多供應商」格局，新供應商首次切入需客戶端電源工程驗證 6-12 個月（高壓 / 漏電流 / 壽命 / 振動測試）+ 量產線小批驗證 3-6 個月；但同類別替代廠商眾多（日系 3 家 + 台廠 5+ 家 + 中國 5+ 家），切換代價遠低於專利或設備鎖定型零件。金山電的議價權主要靠「日系合作 + 一貫化製程」差異化，而非絕對的不可替代——這是其相較立隆電（直達 NVIDIA Rubin GPU 板）戰略地位較低的根因，也是 PE 估值天花板的結構性限制。

1.2 商業模式與營收結構新

1.2.1 商業模式新增

金山電的商業模式：B2B 規格化零件供應 + ODM 通道間接出貨。客戶簽訂年度框架合約（規格 / 數量 / 試壓條件），按月實際拉貨；產品為標準化規格、單顆價格 0.5-15 元、主要靠「合格供應商名單」鎖定生意。收入確認屬「貨物所有權移轉客戶港」（FOB）模式、屬於即期貨款交易（信用 30-60 天），財務透明度較高，無遞延收入或合約資產複雜性。獲利槓桿三層：(1) 產品組合改善（固態占比提升，毛利由 18-22% 拉至 30%+）、(2) 一貫化降本（自製鋁箔 + 電解液對抗原物料漲價）、(3) 規模經濟（泰國三廠開出後攤提固定成本）。
商業模式特徵
▸ 收費模式
單顆計價 · 框架合約 + 月度拉貨

客戶簽訂年度框架合約鎖定規格、數量、試壓條件；實際出貨按月拉貨。液態 0.5-3 元/顆、固態 5-15 元/顆。信用 30-60 天、無遞延收入、無合約資產風險。

▸ 收入確認
FOB（貨物港口移轉所有權）

產品交付客戶指定港口時即認列收入；應收帳款週轉 50-65 天穩定（電子業正常區間）；無「未實現收入」會計風險。月營收公告即時、是判斷基本面變化的最快訊號。

▸ 毛利來源
三層獲利槓桿

(1) 產品組合：固態占比 25% → 35% 拉升、毛利率提升；(2) 一貫化：鋁箔自製 + 電解液自研對抗原物料漲價；(3) 規模：泰國三廠開出後固定成本攤提。三者疊加是 H1 樂觀路徑能成立的基礎。

優勢 vs 脆弱點
▲ 商業模式優勢
  • 規格化 + 多客戶分散：單一客戶風險低、月營收波動相對平穩
  • 一貫化製程降本：銀 / 銅 / 鋁漲價時相對受惠、毛利結構優於外購型同業
  • 固態 25% 切入 AI：透過 ODM 間接出 NVIDIA GB300 / BBU、避免直接 OEM 認證成本
  • 無 CB / 無現增：股本穩定、EPS 不被稀釋（相較同業有結構性優勢）
▼ 商業模式脆弱點
  • 議價權弱：規格化產品、替代廠商眾多（日系 3 + 台廠 5+ + 中國 5+），客戶可快速轉換
  • 規模劣勢：營收僅為立隆電 1/3、缺乏「規模經濟」這一塊護城河（Morningstar 評分 2/5）
  • Tier-2 通道風險：透過 ODM 出貨 = 距離終端 NVIDIA / Hyperscaler 兩層、被「壓價或抽單」風險高
  • 配息率 233% 隱性收割：每年配息高於盈餘、削弱再投資能力與長期成長動能
