8c. 影響鏈推衍
三條關鍵因果鏈
鏈 1：Google TPU 擴張主軸（核心路徑）

正向路徑：Google TPU 擴張 → 資料中心光纖需求增加 → 光聖接單量放大 → 毛利率維持 55%+ → EPS 持續成長

觸發條件Google 維持或擴大 2026 capex 預算（目前 750 億美元）

風險觸發路徑：Google capex 放緩 → 訂單減少 → 營收下滑 → 毛利率因固定成本攤提而壓縮

風險觸發條件Google 下修 capex 超過 15%，或 AI 投資回報不如預期導致策略轉向
鏈 2：CPO 第二曲線（成長期權）

正向路徑：CPO 技術成熟 → 合聖 ELS 產品量產 → 光聖打入 CPO 供應鏈 → 第二成長曲線啟動 → 估值重新評價

觸發條件合聖 ELS 通過客戶驗證、2026H2 開始小批量出貨

阻礙路徑：良率爬坡延遲 → 量產時程推遲 → 市場對 CPO 故事失望 → 估值溢價收斂

阻礙條件ELS 良率未達 80%，或客戶端驗證時間拉長超過 6 個月
鏈 3：估值正常化壓力（逆風路徑）

路徑：PE 88x 為歷史極端值 → 市場情緒轉向或成長放緩 → PE 從 88x 均值回歸至 40-50x → 即使 EPS 維持，股價仍可能回落 30-50%

觸發條件任何一季 EPS 低於市場預期、大盤系統性修正、或 AI 概念退潮

這是最容易被忽視的路徑：好公司 + 壞價格 = 差報酬。歷史上 PE 88x 的公司，即使基本面持續成長，股價在未來 12 個月下跌的機率仍超過 60%。

8c 影響鏈整合結論三條鏈形成「一主一副一逆風」格局。鏈1（Google 主軸）短期穩固但高度依賴單一客戶；鏈2（CPO）是中長期期權，目前尚無法計入估值；鏈3（估值壓力）是最大且最確定的逆風——88 倍 PE 幾乎不容許任何失誤。整體而言，基本面向上力道 vs 估值向下引力，後者在當前價位更具主導性。
