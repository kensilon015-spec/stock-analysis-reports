影響傳播路徑

DRAM 現貨價格大幅上漲 → 短空長多
DRAM原廠漲價 → 宇瞻採購成本上升（佔80-90%）→ 工控端可轉嫁（黏著度高）→ 消費端轉嫁困難 → 毛利率先壓後升
連動標的：2408南亞科（利多）/ 3260威剛（同受影響）/ 2451創見（同受影響）
DRAM 價格崩跌 → 利空
DRAM原廠降價 → 採購成本下降但存貨跌價風險 → 消費端打價格戰 → 毛利率壓力
連動標的：3260威剛 / 4967十銓 / 2451創見
工控/AI邊緣運算需求爆發 → 重大利多
AI伺服器/邊緣運算需求增 → 工控SSD/DRAM模組需求上升 → 宇瞻工控線成長 → 毛利率提升
連動標的：5289宜鼎（同受惠）/ 2395研華（帶動合作）
群聯供應中斷 → 重大利空
群聯佔採購22%為最大供應商 → Flash產品線佔營收61%受衝擊 → 短期營收衝擊顯著
台海地緣政治風險 → 利空（已有因應）
客戶擔憂台灣供應鏈 → 訂單可能轉移非台廠 → 宇瞻已佈局印度+Tier2供應商開發
