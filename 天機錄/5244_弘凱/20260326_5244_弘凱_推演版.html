白話說搬新廠不是萬靈丹。如果客戶在搬遷期間跑掉了、中國競爭者又趁虛而入，那搬完之後的營收不會自動回來。這是最可能發生的劇本。

以 4.86 億/股價 20 元計算，需增發約 2,430 萬股，股本膨脹約 30%。

8g 蝴蝶效應推衍

弱訊號 1（B 類-產業）：Micro LED 聯盟
B 類 弘凱加入錼創 Micro LED 聯盟
如果 Micro LED 在 2027-2028 年大規模商業化，弘凱可能成為供應鏈受益者。但目前 Micro LED 仍處於小量試產階段，距離大規模商用至少還要 2-3 年。

白話說Micro LED 是未來的技術，但「未來」太遠了。弘凱的問題是「現在」的搬廠陣痛和 CB 壓力，等不到 Micro LED 的春天。
弱訊號 2（C 類-治理）：追認轉投資 + 會計師變更
C 類 廣東弘凱追認轉投資 + 即將更換會計師
如果這兩個事件有關聯（例如：會計師發現轉投資問題 → 公司不滿 → 更換會計師），則治理風險可能比表面嚴重得多。追認行為本身就代表內控有缺失：先做了再補程序，而不是先經過董事會核准。
目前無法從公開資訊驗證兩者是否有因果關係，但時間上的巧合值得高度警覺。
白話說先偷偷投資再補報備，這種行為在大公司叫「內控缺失」，在小公司可能暗示更多看不見的問題。加上同時換會計師，讓人不得不多想。
弱訊號 3（A 類-市場）：股價漲停逼近 CB 轉換價
A 類 股價漲停 47.05 元接近 CB 轉換價 46 元
如果有人系統性推升股價促進 CB 轉換，這是一種「解套操作」。一旦推升失敗（股價無法站穩 46 元以上），套牢者和主力可能同時撤退，股價將快速回落。歷史上曾出現股價在 CB 轉換價附近大量換手後暴跌的案例。
觀察重點：46 元附近的成交量是否異常放大、融資餘額是否快速增加。
盲點洞察
盲點 1
廣東弘凱的具體業務和投資金額完全不透明。年報僅有追認揭露，無財務細節。
盲點 2
會計師變更的真正原因無法從公開資訊得知。官方說法通常是「事務所內部調整」，但實際原因可能完全不同。
盲點 3
南通廠的實際良率和稼動率從未公開披露。只能從營收間接推測，存在資訊不對稱。
白話說有三件事是我們完全看不到的：廣東那個轉投資到底花了多少錢在做什麼、為什麼要換會計師、新工廠的良率到底好不好。這三個盲點都可能藏著壞消息。
8h 貓膩推衍
承接第七步貓膩結論：中度可疑（黃燈）
核心假說與反共識偵測
貓膩假說：三重巧合
追認廣東弘凱轉投資 + 會計師變更 + 股價曾推升至 CB 轉換價附近，三者在時間軸上的巧合度過高。如果存在「先做布局 → 推升股價促進 CB 轉換 → 同時為不透明投資安排更順從的會計師」的劇本，則風險遠超表面評估。
嚴重程度：中度可疑（黃燈）。無法證實但無法排除。
歷史劇本比對
樂陞案（2016）相似度 ~20%
樂陞案核心是「推高股價 → 公開收購 → 收購失敗 → 暴跌」。弘凱的情況不同（是 CB 轉換而非公開收購），但「推升股價至特定價位」的動機結構有部分相似。
相似點：小型股、股價有被推升至特定價位的誘因。不同點：弘凱有實體業務、CB 不是收購。
某興櫃公司 CB 案（通用模式）相似度 ~35%
台灣市場常見模式：公司發 CB → 股價無法超越轉換價 → CB 到期前推升股價 → 促進轉換以避免現金償還壓力。這個模式本身不違法，但如果配合內線交易或操縱市場，則構成犯罪。
弘凱目前沒有具體證據指向市場操縱，但「動機」明確存在。
