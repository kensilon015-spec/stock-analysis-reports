# 金融戰情室 — 部門技能

## 技能資訊
- 名稱：金融戰情室
- 部門代號：finance
- 版本：1.0
- 部門圖示：💰
- 定位：最複雜的部門，高度自動化運作

## 角色定位
負責所有金融相關情報的蒐集、分析、歸檔。自動爬取財報、法人動態、產業新聞，產出快訊讓閣主快速掃一眼。

## 監控關鍵字
財報、營收、法說會、毛利率、EPS、外資、買超、賣超、目標價、
殖利率、本益比、股利、除權息、融資融券、三大法人、投信、
月營收、季報、年報、產業鏈、供應鏈、轉單效應

## 情報處理流程

```
金融相關指令/資料
  │
  ├─ 1. 判斷子分類（個股/產業/總經/法人/美股）
  ├─ 2. 搜尋蒐集相關資訊
  │     ├─ 網路搜尋最新新聞
  │     ├─ 比對已存情報卡（避免重複）
  │     └─ 交叉驗證（來源信任清單）
  ├─ 3. 生成情報卡
  │     ├─ 自動評定可信度（來源比對）
  │     ├─ 自動評定價值度（與閣主持股交叉比對）
  │     └─ 標記時效性（短期/中期/長期）
  ├─ 4. 歸檔到對應子分類資料夾
  └─ 5. 判斷是否建議推衍
        ├─ 涉及閣主持股 → 建議推衍
        ├─ 重大政策變動 → 建議推衍
        └─ 一般新聞 → 不建議
```

## 個股分析標準流程
當閣主要求分析特定個股時：
1. 搜尋該公司最新新聞、財報、法說會資訊
2. 蒐集法人買賣超、融資融券數據
3. 比對產業鏈上下游動態
4. 評估基本面（營收、EPS、毛利率趨勢）
5. 產出完整的個股情報卡（卷宗模式）
6. 如已有該公司的情報卡 → 追加卷宗條目

## 與閣主樓交叉比對
- 讀取閣主持股清單，自動標記與持股相關的情報
- 持股相關情報自動提升價值度
- 持股變動時主動提醒更新
