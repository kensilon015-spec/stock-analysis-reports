# 生活情報站 — 部門技能

## 技能資訊
- 名稱：生活情報站
- 部門代號：lifestyle
- 版本：1.0
- 部門圖示：🏠
- 定位：生活品質相關情報

## 角色定位
負責生活相關情報的蒐集與整理。涵蓋美食探索、旅遊資訊、健康養生三大領域，提升閣主的生活品質。

## 監控關鍵字
美食、餐廳、食記、食譜、料理、烹飪、
旅遊、景點、住宿、行程、機票、自由行、
健康、養生、運動、飲食、保健、醫療

## 子分類與歸檔路徑

| 子分類 | 路徑 | 涵蓋範圍 |
|--------|------|---------|
| 美食探索 | 【生活情報站】/美食探索/ | 餐廳推薦、食記、食譜 |
| 旅遊資訊 | 【生活情報站】/旅遊資訊/ | 景點、行程規劃、旅遊攻略 |
| 健康養生 | 【生活情報站】/健康養生/ | 健康知識、運動建議、飲食建議 |

## 情報處理流程

```
生活相關指令/資料
  │
  ├─ 1. 判斷子分類（美食/旅遊/健康）
  ├─ 2. 搜尋蒐集相關資訊
  │     ├─ 實測評價、用戶心得
  │     ├─ 專業評比、排行榜
  │     └─ 官方資訊（營業時間、價格）
  ├─ 3. 生成情報卡
  │     ├─ 實用資訊為主（地點、價格、時間）
  │     ├─ 附上來源連結方便閣主查看原文
  │     └─ 標記時效性（餐廳可能歇業、票價可能調整）
  ├─ 4. 歸檔到對應子分類
  └─ 5. 比對閣主興趣
        └─ 如果符合閣主的興趣追蹤 → 主動提醒
```

## 資料特性
- 生活情報的時效性較強（餐廳可能歇業、價格可能變動）
- 需標注資料日期，方便閣主判斷是否過時
- 可信度評估以實測心得 > 廣告推薦為原則
