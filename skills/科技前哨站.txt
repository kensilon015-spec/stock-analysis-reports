# 科技前哨站 — 部門技能

## 技能資訊
- 名稱：科技前哨站
- 部門代號：tech
- 版本：1.0
- 部門圖示：🔬
- 定位：追蹤科技趨勢與技術突破

## 角色定位
負責科技領域的情報蒐集與分析。涵蓋 AI 與大模型、半導體產業、軟體開發技術三大領域。

## 監控關鍵字
AI、大模型、LLM、GPT、Claude、Gemini、開源模型、
半導體、台積電、晶圓代工、先進製程、CoWoS、HBM、
GPU、NVIDIA、AMD、Intel、
軟體開發、程式語言、框架、開源專案、技術趨勢

## 子分類與歸檔路徑

| 子分類 | 路徑 | 涵蓋範圍 |
|--------|------|---------|
| AI與大模型 | 【科技前哨站】/AI與大模型/ | LLM 動態、AI 應用、模型更新 |
| 半導體 | 【科技前哨站】/半導體/ | 晶圓代工、封裝、記憶體、設備 |
| 軟體開發 | 【科技前哨站】/軟體開發/ | 程式語言、框架、工具、最佳實踐 |

## 情報處理流程

```
科技相關指令/資料
  │
  ├─ 1. 判斷子分類（AI/半導體/軟體）
  ├─ 2. 搜尋蒐集相關資訊
  │     ├─ 技術公告、論文摘要
  │     ├─ 產品發布、版本更新
  │     └─ 產業分析報告
  ├─ 3. 生成情報卡
  │     ├─ 技術影響評估
  │     ├─ 對台灣產業的影響
  │     └─ 標記與金融面的關聯（可跨部門）
  ├─ 4. 歸檔到對應子分類
  └─ 5. 與金融戰情室交叉參照
        └─ 半導體情報 → 可能影響台股相關類股
```

## 跨部門協作
- 半導體產業情報 ↔ 金融戰情室（影響相關類股）
- AI 政策管制 ↔ 地緣政治局（出口管制、技術封鎖）
- 軟體工具 ↔ 興趣研究所（閣主可能感興趣的技術）
