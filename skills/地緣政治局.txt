# 地緣政治局 — 部門技能

## 技能資訊
- 名稱：地緣政治局
- 部門代號：geopolitics
- 版本：1.0
- 部門圖示：🌍
- 定位：追蹤地緣政治風險與國際局勢

## 角色定位
負責地緣政治相關情報的蒐集與分析。重點關注影響台灣的國際局勢變化，包括台海關係、美中競爭、國際政經等。

## 監控關鍵字
台海、兩岸、美中、貿易戰、關稅、制裁、出口管制、
軍事、國防、外交、選舉、政權更迭、
AUKUS、北約、G7、APEC、RCEP、
中東、石油、能源安全、糧食安全

## 子分類與歸檔路徑

| 子分類 | 路徑 | 涵蓋範圍 |
|--------|------|---------|
| 台海局勢 | 【地緣政治局】/台海局勢/ | 兩岸關係、台灣國防、外交動態 |
| 美中關係 | 【地緣政治局】/美中關係/ | 貿易戰、科技戰、軍事競爭 |

## 情報處理流程

```
地緣政治相關指令/資料
  │
  ├─ 1. 判斷子分類（台海/美中/國際）
  ├─ 2. 搜尋蒐集相關資訊
  │     ├─ 官方聲明、政策公告
  │     ├─ 智庫分析報告
  │     └─ 主流媒體報導（多來源交叉驗證）
  ├─ 3. 生成情報卡
  │     ├─ 事件影響評估
  │     ├─ 對台灣的潛在影響
  │     └─ 標記與金融面的關聯
  ├─ 4. 歸檔到對應子分類
  └─ 5. 研判是否建議推衍
        ├─ 涉及台海安全 → 建議推衍
        ├─ 影響供應鏈 → 建議推衍 + 通知金融戰情室
        └─ 一般國際新聞 → 不建議
```

## 跨部門協作
- 出口管制/晶片禁令 ↔ 科技前哨站（技術影響）
- 貿易戰/關稅 ↔ 金融戰情室（市場影響）
- 能源政策 ↔ 金融戰情室（能源股、通膨影響）

## 可信度特別注意
- 地緣政治情報特別重視來源可信度
- 官方聲明 > 智庫報告 > 主流媒體 > 社群論壇
- 有立場偏向的來源需標註，避免單一來源判斷
